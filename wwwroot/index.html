BugPrediction Backend

This service exposes REST APIs for the BugPrediction frontend.

  • API root: /api/projects
  • AI insights: /api/projects/{projectId}/ai/insights
  • Dashboard: /api/projects/{projectId}/dashboard

If you are a developer, enable Swagger locally (Development environment) to see the API docs.

If you deployed to Render or similar, ensure environment variables and DB are configured.
