## 2. T-SQL → PL/pgSQL Conversion Rules

| Legacy Platform | PostgreSQL | Notes |
|---|---|---|
| `CREATE PROCEDURE sp_Name` | `CREATE OR REPLACE FUNCTION dbo.sp_name()` | All lowercase in PostgreSQL |
| `@Parameter INT` | `p_parameter INT` | Prefix `p_` to all params |
| `NVARCHAR(n)` | `VARCHAR(n)` | Direct replacement |
| `DATETIME` | `TIMESTAMP` | No timezone |
| `DECIMAL(5,2)` | `NUMERIC(5,2)` | Direct replacement |
| `BIT` | `BOOLEAN` | Direct replacement |
| `TOP 1` | `LIMIT 1` | |
| `TOP (@N)` | `LIMIT p_n` | |
| `SET NOCOUNT ON` | *(remove — not needed)* | |
| `SCOPE_IDENTITY()` | `RETURNING id INTO v_newid` | Use DECLARE variable |
| `IF EXISTS BEGIN...END` | `IF EXISTS THEN...END IF` | |
| `SELECT CAST(SCOPE_IDENTITY() AS INT)` | `RETURN QUERY SELECT v_newid` | |
| Result set via SELECT | `RETURNS TABLE(...) + RETURN QUERY` | |
| No return | `RETURNS VOID` | |

### Critical PostgreSQL Rules
1. **Always cast `NOW()` as `NOW()::TIMESTAMP`** when calling functions from pgAdmin — `NOW()` returns `TIMESTAMPTZ` which causes type mismatch
2. **All function names are lowercase** in PostgreSQL — `sp_GetAllProjects` becomes `sp_getallprojects`
3. **After bulk COPY import, always sync sequences:**
   ```sql
   SELECT setval(pg_get_serial_sequence('dbo.projects', 'id'), (SELECT MAX(id) FROM dbo.projects));
   -- repeat for all 7 tables
   ```
4. **`dbo` schema must be created first:** `CREATE SCHEMA IF NOT EXISTS dbo;`
5. **Foreign keys reference table without schema prefix in FK declaration** but use schema prefix everywhere else

## 4. C# ADO.NET Calling Conventions

### Connection Setup (`appsettings.json`)
```json
"ConnectionStrings": {
  "NeonDb": "Host=your-neon-host;Database=bpcqdb;Username=xxx;Password=xxx;SSL Mode=Require"
}
```

### DI Registration (`Program.cs`)
```csharp
builder.Services.AddScoped<BpcqDbService>(_ =>
    new BpcqDbService(builder.Configuration.GetConnectionString("NeonDb")!));
```

### Calling GET functions
```csharp
// Returns List<T>
using var cmd = new NpgsqlCommand("SELECT * FROM dbo.sp_getallprojects()", conn);

// Returns single row
using var cmd = new NpgsqlCommand("SELECT * FROM dbo.sp_getprojectbyid(@p_projectid)", conn);
cmd.Parameters.AddWithValue("p_projectid", projectId);
```

### Calling INSERT functions (returns id)
```csharp
using var cmd = new NpgsqlCommand("SELECT id FROM dbo.sp_insertsnapshot(@p_projectid, ...)", conn);
var newId = Convert.ToInt32(await cmd.ExecuteScalarAsync());
```

### Calling VOID functions
```csharp
using var cmd = new NpgsqlCommand("SELECT dbo.sp_insertseveritydistribution(@p_snapshotid, ...)", conn);
await cmd.ExecuteNonQueryAsync();
```

### NULL handling
```csharp
// Always use DBNull.Value for nullable params — never pass C# null directly
cmd.Parameters.AddWithValue("p_organization", (object?)req.Organization ?? DBNull.Value);

// Always check IsDBNull when reading nullable columns
Organization = r.IsDBNull(r.GetOrdinal("organization")) ? null : r.GetString(r.GetOrdinal("organization"))
```

### Timestamp handling
```csharp
// DateTime in C# maps cleanly to TIMESTAMP in PostgreSQL via Npgsql
// No special casting needed in code — only pgAdmin needs NOW()::TIMESTAMP
cmd.Parameters.AddWithValue("p_scandate", req.ScanDate); // works fine
```

## 5. Data Type Mapping — C# ↔ PostgreSQL

| PostgreSQL Type | C# Type | Reader Method |
|---|---|---|
| INT | `int` | `reader.GetInt32()` |
| BIGINT | `long` | `reader.GetInt64()` |
| VARCHAR | `string` | `reader.GetString()` |
| BOOLEAN | `bool` | `reader.GetBoolean()` |
| TIMESTAMP | `DateTime` | `reader.GetDateTime()` |
| NUMERIC(5,2) | `decimal` | `reader.GetDecimal()` |
| NULL any type | `null` | check `reader.IsDBNull()` first |

- [ ] Create `dbo` schema: `CREATE SCHEMA IF NOT EXISTS dbo;`
- [ ] Run `01_CreateDatabase_Tables.sql` — creates all 7 tables
- [ ] Run `02_StoredProcedures_PostgreSQL.sql` — creates all 22 functions

- [ ] **Sync all sequences after COPY import** (critical — prevents duplicate key errors)

## 7. Known Gotchas

| Issue | Cause | Fix |
|---|---|---|
| `duplicate key value violates unique constraint` | Sequence out of sync after COPY import | Run `setval` for all 7 tables |
| `function does not exist` type error | `NOW()` returns TIMESTAMPTZ, function expects TIMESTAMP | Use `NOW()::TIMESTAMP` in pgAdmin |
| `unknown` argument type error | String literals passed without cast in pgAdmin | Add `::VARCHAR` cast to string args |
| Duplicate functions in schema | Previously existed with different naming | DROP old ones, keep matching original SP names |
| Function name not found | PostgreSQL lowercases all identifiers | Always use lowercase: `sp_getallprojects` not `sp_GetAllProjects` |
